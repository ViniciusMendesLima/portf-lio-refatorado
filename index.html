Habilidades

HTML5

HTML é a linguagem de marcação usada para estruturar páginas da web. Ele define elementos como títulos, parágrafos, imagens, links e tabelas, organizando o conteúdo para exibição nos navegadores.

CSS3

CSS é a linguagem usada para estilizar páginas da web, definindo cores, fontes, layouts e responsividade. Ele permite separar a estrutura (HTML) da apresentação, tornando o design mais flexível e eficiente.

GitHub

GitHub é uma plataforma de hospedagem de código que facilita o controle de versão e a colaboração em projetos usando Git

Git

Git é um sistema de controle de versão distribuído que permite gerenciar e rastrear alterações em projetos de software.
